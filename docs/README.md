## 📊 关键数据摘要

### 市场规模
- 2024年具身智能市场规模：**8000亿元**
- 2026年预计规模：**突破1万亿**
- 2025年具身智能数据服务市场：**约143亿元**

### 全球竞争
- 中国厂商占全球人形机器人出货量：**超70%**
- 2025年全球人形机器人出货量：**1.3-1.8万台**
- 中国企业全球前三：智元、宇树、优必选

### 投融资
- 2025年具身智能融资总额：**约400亿元**
- 同比增长：**3倍**
- 融资事件数量：**329-370起**

### 技术突破
- Real2Sim成本降低：**10元/条 → 0.02元/条**（降低500倍）
- Open X-Embodiment数据集：**100万+轨迹**
- 中国开发者开源模型下载量：**占全球17%**
